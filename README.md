Gestion des inscriptions

Objectif:
Projet réalisé dans le cadre de l’apprentissage du développement web dynamique avec PHP et base de données.

---

Description:
Application web permettant de gérer les inscriptions à une course.

Cette application permet d’ajouter, modifier et supprimer des participants, ainsi que de gérer leurs informations via une base de données.

---
🚀 [Démo Live](https://inscription.nzabidesigns.fr)

--
 Fonctionnalités:
 
- Ajouter un participant
- Supprimer un participant(Admin)
- Modifier les informations d’un participant(Admin)
- Choisir un maillot
- Afficher la liste des participants

---

Technologies:
- PHP
- HTML / CSS
- SQL (base de données)

---

Base de données
- Utilisation de requêtes SQL
- Récupération et gestion des données (SELECT, INSERT, UPDATE, DELETE)
